FAQ'S
What is Nexa?
Nexa is a simple e-commerce website where users can browse and view tech accessories and stationery products. It was built as a frontend project to practice web development skills.
Can I buy products directly from the website?
Not yet. Nexa is currently a frontend-only project, so payments and real orders are not supported at the moment.
Does Nexa have a shopping cart?
Yes. Users can add items to a cart, but the functionality is basic and limited as the site is not connected to a real backend or database.
Is the website mobile-friendly?
Yes. Nexa is designed to be responsive, so it works on mobile, tablet, and desktop screens.
What technologies were used to build Nexa?
Nexa was built using HTML, CSS, and JavaScript without any frameworks.
Will more features be added in the future?
Yes. Planned improvements include adding a backend, user authentication, a database and a complete checkout system including real products.
Can I use or modify this project?
Yes. This project is open for learning and practice. You can fork it, modify it, and build on top of it.
Do I need an account to use Nexa?
You can browse products without an account, but Nexa includes a simulated authentication system for login and signup features.
