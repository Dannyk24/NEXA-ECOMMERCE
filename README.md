# 🛍️ Nexa

Nexa is a simple e-commerce website for selling tech accessories and stationery.

This project was built to practice frontend development using HTML, CSS, and JavaScript.

---

## 🚀 What it does

* Displays products
* Shows prices and basic details
* Simple add to cart
* Clean and responsive layout
* User wishlist system
* Product search System
* Simluted authentication and data storage system including data persistence

---

## 🧑‍💻 Built With

* HTML%
* CSS
* JavaScript
* Handbrake(for video optimization)
